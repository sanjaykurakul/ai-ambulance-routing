# AmbulanceAI - Smart Ambulance Routing System

## About Project

AmbulanceAI is a front-end web application developed using HTML, CSS and JavaScript. The purpose of this project is to demonstrate how AI-based route optimization can help ambulances reach patients and hospitals faster during emergencies.

The website provides information about the system, its features, workflow, pricing plans, client feedback, and a contact form. This project is created as a portfolio and learning project.

## Features

* Responsive design for mobile and desktop
* Light/Dark mode
* Smart route optimization concept
* Hospital availability information
* Live ambulance tracking interface
* Animated statistics section
* Client feedback section
* Contact form validation
* API integration using Fetch API

## Technologies Used

* CSS3
* JavaScript (ES6)
